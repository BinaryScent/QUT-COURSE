# 资料结构
* 课程放在对应的开课学院下，所以先确定自己的开课学院，再确定课程名
* 课程目录结构如下：
   ```
   course(课程名)
   ├ homeworks(作业)
   ├ labs(实验)
   ├ exams(考试)
   ├ notes(笔记)
   └ README.md
   ```

## 投稿
- 仅接受学生原创的或者获得授权的资源
